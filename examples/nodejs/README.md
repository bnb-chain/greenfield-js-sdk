# GreenField JS SDK Node.js


## Prepare

### install and build

Install dependencies:

```bash
pnpm install
```

Build package:

```bash
pnpm run -F "./packages/**"  -r build
```

### Run the demo application

Fill your account's private key and address in [env.js](./env.js).

## Execure example

```bash
> node storage.js
```
