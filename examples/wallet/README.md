# GreenField JS SDK Example

Use [React](https://react.dev/) and [Next.js](https://nextjs.org/)

## Getting Started

```bash
# DEV ENV
> cp .env.simple .env.development.local

# QA ENV
> cp .env.simple .env.test.local
```

## Install and Build

First, build SDK srouce code:

```bash
# root dir
> pnpm run install
> pnpm run -r build 
```

## Start

```bash
# dev ENV
> npm run dev
```

```bash
# qa env
> npm run dev:qa
```
