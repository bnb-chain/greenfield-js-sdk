1. Modern browsers (Chromium, Firefox, Safari)
2. Browser extensions (Chromium, Firefox)
3. Nodejs

## Contribution

1. install dependencies

```bash
> pnpm install
```

2. Build package:

```bash
> pnpm run build
```
